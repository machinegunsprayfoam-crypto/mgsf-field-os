🔗 INTEGRATIONS — ONE WEBHOOK WIRES KLYFTON TO EVERYTHING
Klyfton is a self-contained app, so it reaches the outside world through one Event Webhook. Point that webhook at a free Zapier or Make "Catch Hook," and from there you can fan out to 6,000+ apps — text yourself, email the customer, drop jobs on Google Calendar, push deals to HubSpot, create QuickBooks invoices, ask for reviews. Set it up once.
  1. In Zapier, create a Zap → trigger Webhooks by Zapier → Catch Hook. Copy the URL it gives you.
  2. Add an action for what you want: SMS by Zapier, Gmail, Google Calendar, HubSpot, QuickBooks, etc.
  3. In Vercel → mgsf-fieldos → Settings → Environment Variables, add ALERTS_WEBHOOK_URL = that URL, then Redeploy.
  4. Filter each Zap by the event field so the right action fires for the right event (below).
Events Klyfton fires (the event field):
• new_lead — web quote submitted → text/email you, add to HubSpot/Sheets
• job_scheduled — job created → add to Google Calendar, text the crew
• job_completed — job marked Invoiced → notify office, start closeout
• invoice — job Invoiced → create the QuickBooks/PayPal invoice
• review_ask — job marked Paid → send the Google review request
• reorder — inventory hit its reorder point → email the supplier
Each event also carries name, phone, email, service, address, value, customer, job_date & a ready-to-send message string.
